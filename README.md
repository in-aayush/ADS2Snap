
# 🚀 ADS2Snap – Offline Advertisement Generator

## 📌 Overview
ADS2Snap is an offline AI-powered system that generates advertisement videos from product images using Python. The system simulates AI behavior without using any external APIs.

---

## 🎯 Features

- 🧠 AI-like script generation (rule-based NLP)
- 🎨 Color extraction from images
- 🎬 Multi-scene video ad creation
- 🔊 Offline voice generation (pyttsx3)
- 🎵 Background music integration
- 📝 Subtitle generation
- 📊 Ad scoring system
- 🖥️ Streamlit UI dashboard
- 📦 Fully offline (No API)

---

## 🏗️ Architecture

Image → Category Detection → Script → Voice → Music → Video → Output

---

## 🛠️ Technologies Used

- Python
- MoviePy
- OpenCV
- Streamlit
- pyttsx3
- NumPy
- Pillow
